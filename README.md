# Project Name: FrontRowTeam CasinoApp by Philip Miethke, Oscar Rosas, and Brendan Lockwood
# Project Descripition:
Our Project is a Casino/Card based program that allows you to play both BlackJack and a Simplified version of Poker. You can Bet an amount from your pool of chips and bet them as you play. You have the Choice to hit, stand, fold and bet. You play against a a singular other player can freely choose to switch between the two games at anytime. The main inspiration for this project came from our personal experiences playing card games with family and the classic computer card games like solitaire and so on.
